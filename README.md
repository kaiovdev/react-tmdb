# 🎬 React TMDB

Aplicação web desenvolvida em **React** que consome a API do TMDB (The Movie Database) para exibir filmes, detalhes e salvar filmes como favoritos.

## 🚀 Funcionalidades

- 📄 Listagem de filmes populares
- 🔍 Visualização de detalhes de cada filme
- 🎞️ Exibição de imagens, títulos e informações
- 📱 Layout responsivo
- ⭐ Salvas filmes favoritos

## 🛠️ Tecnologias utilizadas

- React
- React Router DOM
- Axios
- CSS modules

## 🔑 Integração com a API do TMDB

Este projeto utiliza a API oficial do TMDB.

👉 Para acessar a documentação da API:
https://developer.themoviedb.org/docs/getting-started

---

## ⚙️ Configuração do ambiente

Para rodar o projeto localmente, você precisa criar um arquivo `.env` na raiz do projeto e adicionar sua chave da API do TMDB.

### 1. Crie o arquivo `.env`

```bash
.env
```

### 2. Adicione sua chave da API

VITE_API_TOKEN=SEU_TOKEN

### COMO RODAR O PROJETO:

# Instalar dependências

npm install

# Rodar o projeto

npm run dev
